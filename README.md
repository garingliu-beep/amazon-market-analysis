# Amazon 选品市场分析工具

> 专业的亚马逊选品市场分析工具，从关键词、ASIN、消费场景三个入口出发，输出决策级市场调研报告。

## 功能特性

- **三种入口方式** — 关键词入口（品类方向）、ASIN入口（反推市场）、消费场景入口（挖掘机会）
- **三种调研深度** — 探索级（AI独立完成）、验证级（用户部分数据+AI补充）、决策级（完整数据+深度加工）
- **决策引擎** — 供需指数模型、可进入空间公式、广告压力公式、正式机会判定5条件
- **避坑信号** — 头部锁死 / 广告吞利润 / 产品不可分层，快速排除不值得做的市场
- **HTML 报告** — 完整版 / 精简版 / 会议四页版（老板决策用）
- **AI 边界清晰** — 硬指标规则计算，AI 只负责解释和补充，报告标注推断来源

## 核心方法论

基于「媛媛方法论」选品决策框架：

1. **前置风险排除** — 侵权排查 + 认证合规检查（红线）
2. **避坑信号识别** — 快速排除不值得做的市场
3. **核心因素分析** — 客单价 / 竞品毛利 / 广告压力 / 市场体量 / 竞争度 / 盈利周期
4. **可进入空间计算** — 市场容量 × 非垄断空间 × 可运营性 × 利润余量
5. **综合决策矩阵** — 8 维度加权评分

## 三种调研模式

| 模式 | 名称 | 用户提供 | AI 负责 | 适用场景 |
|------|------|---------|---------|---------|
| 模式一 | 探索级 | 仅关键词或 ASIN | 全流程独立调研 | 初步了解品类 |
| 模式二 | 验证级 | 部分数据 | 补充+交叉验证 | 已有初步数据，需验证 |
| 模式三 | 决策级 | 完整数据 | 深度加工+决策建议 | 最终决策前的深度分析 |
