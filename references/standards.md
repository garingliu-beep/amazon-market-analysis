## 竞争格局判断标准

### 综合判断表

| 指标 | 友好 | 中等 | 困难 |
|------|------|------|------|
| HHI指数 | < 1500 | 1500-2500 | > 2500 |
| Top10集中度 | < 40% | 40-60% | > 60% |
| 头部平均Review | < 1000 | 1000-3000 | > 3000 |
| 新品存活率 | > 20% | 10-20% | < 10% |

### 细化竞争指标

**Review壁垒细化：**
- 至少8个链接评论<250 → 评论壁垒不高，新卖家有机会
- 首页全是千评以上、评分4.7+的老链接 → 壁垒很高，新手慎入
- 低评论新品能稳定出单 → 真正的新卖家机会

**品牌集中度细化：**
- 第一品牌占比<25% → 市场未被垄断
- 前三品牌合计<55% → 有竞争空间
- 第一品牌占比>40% 或 前三品牌合计>70% → 高度垄断，慎入

**配送时效竞争：**
- 首页全是FBA次日达 → 配送竞争激烈
- 有FBM或较长配送时间的产品也在出单 → 配送不是核心竞争维度

**价格竞争：**
- 首页价格差异大（>30%）→ 有定价空间
- 首页价格高度一致 → 价格战激烈

### 推广成本评估矩阵

| 指标 | 低竞争（绿灯） | 中等竞争（黄灯） | 高竞争（红灯） |
|------|--------------|----------------|--------------|
| 核心词CPC | < $1.0 | $1.0-$2.0 | > $2.0 |
| 首页平均评论数 | < 200 | 200-500 | > 500 |
| 首页平均评分 | < 4.3 | 4.3-4.5 | > 4.5 |
| 品牌集中度HHI | < 1500 | 1500-2500 | > 2500 |

**综合判断**：
- 全部绿灯 → 推广成本低，适合新手
- 有黄灯但无红灯 → 有一定推广成本，需有策略
- 有红灯 → 推广成本高，需谨慎

**CPC获取方式**：
- 最准确：在亚马逊后台开小额广告活动，查看系统建议竞价范围
- 第三方工具：卖家精灵、Helium 10等（有延迟，仅供参考）

## 供需指数计算与判定标准（新增）

### 计算公式

```
该组合Listing销量占比 = 该组合的总销量 / 全部样本总销量
该组合Listing数量占比 = 该组合的产品数 / 全部样本产品数
供需指数 = 销量占比 ÷ 数量占比
```

### 判断标准

| 供需指数 | 含义 | 业务动作 |
|---------|------|----------|
| > 1.5 | 需求显著强于供给 | 优先机会，重点开发 |
| 1.0-1.5 | 需求略强于供给 | 有机会，需差异化 |
| = 1.0 | 供需基本匹配 | 竞争均衡，需差异化突围 |
| < 1.0 | 供给相对多于需求 | 慎入，除非有强差异化 |

### 正式机会判定（5条件必须同时满足）

| 条件 | 标准 | 说明 |
|------|------|------|
| 1. 样本过线 | Listing数量 ≥ 10 | 不是1-2个样本算出来的高指数，避免小样本偏差 |
| 2. 效率过线 | 供需指数 > 1 | 销量占比明显高于数量占比 |
| 3. 平均销量 | 高于品类中位数的50% | 有真实需求支撑，不是虚假繁荣 |
| 4. 产品过线 | 组合维度有明确产品意义 | 能被供应链理解和打样，能对应到具体产品 |
| 5. 卖点可表达 | 能形成卖点表达 | 能写进Listing标题和A+页面 |

**不满足全部5条件的 → 观察项，不直接立项**

### 供需指数应用场景

| 应用场景 | 说明 |
|---------|------|
| 模式一 Step 6 | 基于搜索结果初算，精度★★☆☆☆ |
| 模式二 Step 5 | 基于用户数据计算，精度★★★★☆ |
| 模式三 Step 4 | 基于完整数据深度分析，精度★★★★★ |

## 维度评分标准（新增）

### 维度评分公式

```
维度得分 = 搜索需求×35% + 区分产品×20% + 购买影响×20% + 可打标×15% + 业务可用×10%
```

### 各因子评分标准（1-10分）

**搜索需求（权重35%）** — 买家是否真的在搜这个维度的词

| 分数 | 标准 |
|------|------|
| 9-10 | 该维度相关关键词月搜索量 > 10000，多个词有稳定搜索 |
| 7-8 | 该维度相关关键词月搜索量 3000-10000 |
| 5-6 | 该维度相关关键词月搜索量 1000-3000 |
| 1-4 | 该维度相关关键词月搜索量 < 1000，或搜索量在下降 |

**区分产品（权重20%）** — 这个维度能否分出不同的产品层

| 分数 | 标准 |
|------|------|
| 9-10 | 该维度能清晰分出3+个产品层，各层之间有明显差异 |
| 7-8 | 能分出2-3个产品层，有一定差异 |
| 5-6 | 能分出2个产品层，但差异不大 |
| 1-4 | 所有产品在这个维度上几乎一样，无法区分 |

**购买影响（权重20%）** — 这个维度是否影响买家的购买决策

| 分数 | 标准 |
|------|------|
| 9-10 | 买家评论中高频提及该维度，是核心决策因素（如材质、安全性） |
| 7-8 | 买家评论中中频提及，是重要因素（如尺寸、功能） |
| 5-6 | 买家评论中低频提及，是次要因素（如颜色、包装） |
| 1-4 | 买家几乎不关注该维度 |

**可打标（权重15%）** — 能否对每个产品标注这个维度的值

| 分数 | 标准 |
|------|------|
| 9-10 | 能从标题/描述中明确识别该维度的值（如材质=不锈钢） |
| 7-8 | 大部分产品能识别，少数需要推断 |
| 5-6 | 约一半产品能识别 |
| 1-4 | 大部分产品无法从标题/描述中识别该维度的值 |

**业务可用（权重10%）** — 这个维度能否指导选品和开发

| 分数 | 标准 |
|------|------|
| 9-10 | 能直接形成产品方向和卖点（如"不锈钢材质→开发不锈钢版本"） |
| 7-8 | 能辅助产品决策（如"大号需求多→主推大号"） |
| 5-6 | 有一定参考价值但不能直接指导开发 |
| 1-4 | 无法转化为产品行动 |

### 维度筛选标准

| 得分 | 判断 | 动作 |
|------|------|------|
| ≥ 7.0 | 强维度 | 必须纳入分析 |
| 5.0-6.9 | 弱维度 | 可选纳入 |
| < 5.0 | 无效维度 | 排除 |

### 维度发现流程

```
核心词（流量前90%）
    ↓
词根（去主词和泛词）
    ↓
AI聚类（按语义相似性分组）
    ↓
五维评分（搜索需求+区分产品+购买影响+可打标+业务可用）
    ↓
AI补全（根据品类常识补充可能遗漏的维度）
    ↓
人工确认（用户选3-6个最终维度）
```

## 可进入空间计算标准（新增）

### 计算公式（乘法模型）

```
可进入空间 = 市场容量因子 × 非垄断空间因子 × 可运营性因子 × 利润余量因子
```

### 各因子取值（0-1）

**市场容量因子**：

| 条件 | 取值 |
|------|------|
| 月GMV > $500K | 1.0 |
| 月GMV $200K-500K | 0.7 |
| 月GMV $100K-200K | 0.5 |
| 月GMV < $100K | 0.2 |

**非垄断空间因子**：

| 条件 | 取值 |
|------|------|
| Top10集中度 < 40% | 1.0 |
| Top10集中度 40-60% | 0.7 |
| Top10集中度 60-70% | 0.4 |
| Top10集中度 > 70% | 0.1 |

**可运营性因子**：

| 条件 | 取值 |
|------|------|
| 推广成本评估全绿灯 + 新品存活率>20% | 1.0 |
| 有黄灯但无红灯 + 新品存活率10-20% | 0.6 |
| 有红灯 或 新品存活率<10% | 0.2 |

**利润余量因子**：

| 条件 | 取值 |
|------|------|
| 基准情景净利率 > 25% | 1.0 |
| 基准情景净利率 20-25% | 0.7 |
| 基准情景净利率 15-20% | 0.4 |
| 基准情景净利率 < 15% | 0.1 |

### 判断标准

| 可进入空间 | 判断 | 动作 |
|-----------|------|------|
| > 0.3 | 值得进入 | 可以投入资源 |
| 0.15-0.3 | 有条件进入 | 需要差异化策略 |
| < 0.15 | 不建议 | 转向其他市场 |

### 乘法模型特性

> 乘法模型的优势：任一因子极差（如0.1），整体结果都会被拉低。这比加权打分更能暴露致命短板。
>
> 示例：市场容量1.0 × 非垄断0.1 × 可运营1.0 × 利润1.0 = 0.1（不建议），即使其他三个维度都很好。

## 广告压力判断标准（新增）

### 核心公式

```
ACOS = PPC ÷ (转化率 × 平均售价)
```

### 三变量联动分析

只看 CPC 高低会误判，必须三个变量一起看：

| 变量 | 高 | 低 |
|------|------|------|
| PPC | 拿流量贵 | 拿流量便宜 |
| 转化率 | 流量容易成交 | 流量难成交 |
| 平均售价 | 广告容错空间大 | 广告容错空间小 |

### 广告压力判断

```
广告费占比 = ACOS × (1 - 退货率)
广告承压空间 = 毛利率 - 广告费占比
```

| 广告承压空间 | 判断 | 说明 |
|-------------|------|------|
| > 15% | 广告压力小 | 有充足推广空间 |
| 10-15% | 广告压力中等 | 需精细运营控制ACoS |
| 5-10% | 广告压力大 | 利润被压缩，需谨慎 |
| < 5% | 广告压力极大 | 基本没有利润，不建议 |

### ACOS与毛利率关系

| ACOS / 毛利率 | 判断 |
|---------------|------|
| < 40% | 推广空间充足 |
| 40-60% | 需要精细运营 |
| > 60% | 利润被严重压缩 |

## 特征归一化规范（新增）

### 正确合并原则

- **只合并消费者理解上等价的表达**
- 流程：原始识别值 → 标准词义 → 上位特征组
- 保留主流特征和长尾特征的区分
- 示例：
  - "stainless steel" + "SS" + "金属" → 标准词义"不锈钢" ✅
  - "BPA-free" + "food grade" → 不合并（决策含义不同） ✅

### 错误合并（禁止）

| 错误类型 | 示例 | 为什么错 |
|---------|------|----------|
| 低占比强行塞进主流类 | 把"竹制"（3%）合并到"塑料"（60%） | 不同材质，决策含义完全不同 |
| 不同决策含义混在一起 | 把"防水"和"耐用"归为同一维度 | "防水"是功能，"耐用"是质量，买家分开考虑 |
| 凑样本牺牲产品意义 | 把"户外用"和"家用"合并为"通用" | 场景不同，产品规格要求不同 |

### 归一化检查清单

1. 合并后的标准词义是否为消费者理解上的等价表达？
2. 合并后是否会丢失产品差异信息？
3. 合并后的组是否能对应到具体的产品方向？
4. 如果答案是否，不要合并。

## 品牌集中度计算方法

### HHI指数计算

1. 按品牌聚合所有ASIN的月销量和月销额
2. 计算各品牌市场份额（%）= 品牌月销量 / 品类总月销量 × 100
3. HHI = Σ (各品牌市场份额%)²

**判断标准：**
- HHI < 1500：分散市场（对新卖家友好）
- HHI 1500-2500：中等集中（有机会但需策略）
- HHI > 2500：高度集中（入场需谨慎）

### 头部集中度计算

```
Top10集中度 = TOP10月销量 / TOP_N月销量 × 100%
```

**判断标准：**
- \> 60%：高度垄断
- 40-60%：中度集中
- < 40%：分散市场

## 流量结构分析方法

### 关键词集中度分析

```
关键词集中度 = Top 10关键词月搜索量 / 总关键词月搜索量 × 100%
```

**判断标准：**
- \> 60%：高集中度，流量被头部垄断
- 30-60%：中等集中度，有一定长尾机会
- < 30%：低集中度，流量分散，长尾词机会多

### 流量可获取性评分（1-10分）

| 分数 | 条件 |
|------|------|
| 9-10 | 关键词分散、长尾词多、头部未锁定流量入口 |
| 7-8 | 有一定集中度但仍有长尾机会 |
| 5-6 | 较集中，需要差异化切入 |
| 1-4 | 高度垄断，流量入口被头部锁定 |

## 价格敏感度判断标准

### 价格与销量相关性分析

| 相关系数 | 敏感度 | 含义 | 竞争策略 |
|---------|--------|------|----------|
| r < -0.6 | 高 | 价格敏感，降价能显著提升销量 | 价格战有效，但利润受限 |
| -0.6 ≤ r < -0.3 | 中 | 有一定价格敏感性 | 适度价格竞争 |
| -0.3 ≤ r ≤ 0.3 | 低 | 价格不敏感，竞争在其他维度 | 品牌/功能/Review驱动 |
| r > 0.3 | 反向 | 高价高销，品牌/品质驱动 | 差异化定价 |

## 综合评分标准

| 分数区间 | 建议行动 | 说明 |
|---------|---------|------|
| ≥ 8.0 | 强烈建议入场 | 多维度优秀，短板不明显 |
| 6.5-7.9 | 建议入场 | 整体良好，需关注个别短板 |
| 5.0-6.4 | 谨慎考虑 | 存在明显短板，需进一步验证 |
| < 5.0 | 不建议 | 多维度不达标 |

### 各维度评分细则（1-10分）

**市场需求（权重25%）**
- 9-10：月销大且增长中
- 7-8：月销中等且稳定
- 5-6：月销偏小或下降中
- 1-4：月销很小且衰退

**竞争可行性（权重25%）**
- 9-10：分散市场、低壁垒、新品友好
- 7-8：中等集中、中等壁垒
- 5-6：较集中、较高壁垒
- 1-4：高度垄断、高壁垒

**利润空间（权重20%）**
- 9-10：净利率>25%且单件利润>$8
- 7-8：净利率20-25%且单件利润$5-8
- 5-6：净利率15-20%且单件利润$3-5
- 1-4：净利率<15%或单件利润<$3

**差异化空间（权重20%）**
- 9-10：多个正式机会（供需指数>1）且竞品未覆盖
- 7-8：有正式机会但需验证
- 5-6：差异化空间有限
- 1-4：产品高度标准化，难以差异化

> **模式一特殊处理**：探索级调研下，差异化空间维度因数据不足难以精确评分。权重调整为：市场需求30%、竞争可行性30%、利润空间20%、差异化空间10%、风险可控度10%。

**风险可控度（权重10%）**
- 9-10：无明显风险
- 7-8：有风险但可控
- 5-6：有中等风险需注意
- 1-4：高风险因素存在
